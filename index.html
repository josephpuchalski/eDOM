easyDOM

A lightweight and streamlined JavaScript DOM program that allows users to make changes to the DOM easily with less lines of code.

Click the buttons below for some simple examples of how easyDOM works. I have included the corresponding code that manipulates the DOM. All of the buttons also use eDOM "on" click function. For more details, see the documentation.

eDOM("li").addClass("red")

eDOM("li").removeClass("red")

eDOM("li").html("hello")

eDOM("li").append(" 2017")

eDOM("li").empty()

eDOM("li").remove()
